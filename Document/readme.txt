HoloeverMR SDK Pro Unity开发包：

请开发者注意：

HoloeverMR  SDK面向Holoever VR/AR设备设计，请使用VR/AR设备进行开发和调试。

不支持手机运行。

****************************************

包含内容：

1，Demo：提供Unity演示Demo，展示如何使用XR SDK插件

2，Plugins：XR SDK Unity插件，需要导入到Unity项目中使用

3，文档：开发使用文档

4，HoloeverMR 启动器：NibiruXR应用启动器，在手机中可用于正常启动XR SDK应用。一体机无需安装。
