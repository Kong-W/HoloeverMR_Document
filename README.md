## HoloeverMR SDK 导入说明

### 1、新建工程
### 2、点击 Windows/PackageManager,打开
### 3、点击左上角+ 选择 [add package from git URL]
### 4、填入地址 https://github.com/Kong-W/HoloeverMR.git
### 5、点击[add]
### 6、点击左下角刷新按钮
### 7、等待加载完成
### 8、找到 [Package/HoloeverXR For Unity/Holoever/UnityPackage/HoloeverMR Assets] 双击导入资源
### SDK导入完成
